Challenges & Lessons Learned

Dependency Management

Challenge: Dependency issues with OpenSlide, BioFormats, and Java bindings; complex container compatibility.

Solution: Built multi-stage Dockerfiles to isolate dependencies and ensure reproducible, unified environments.

GPU Runtime Compatibility

Challenge: CUDA library and driver version mismatches causing GPU initialization failures across host machines.

Solution: Standardized on pinned NVIDIA NGC base images with strictly defined PyTorch and CUDA versions.

Large WSI Data Handling

Challenge: Handling massive WSI arrays efficiently without causing severe I/O hotspots or memory limits.

Solution: Implemented chunked streaming I/O, robust LRU cache strategies, and memory-aware worker pooling.

GeoJSON Optimization

Challenge: Multi-megabyte GeoJSON sizes for dense cell segmentations crashing the web browser.

Solution: Applied dynamic vector tiling, topology-preserving decimation, and gzip compression for web delivery.
