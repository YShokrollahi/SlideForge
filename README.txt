# VitaminPScope slide HTML set

These slides are designed for the SlideForge HTML-to-PPT workflow.

## Files
- `shared.css`: common styling
- `slides/*.html`: one HTML file per slide

## Notes
- Canvas is fixed at 1280x720
- Placeholder panels can be replaced with screenshots or figures later
- Relative path to CSS assumes each slide sits in `slides/`
